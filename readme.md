## 🛒 ShopEase - Responsive E-Commerce Website
## Live Demo:
**Live Website:** https://meghanadoguparthi-10.github.io/ShopEase/
**📌 Overview** 

ShopEase is a responsive e-commerce website developed using HTML, CSS, and JavaScript. The project provides a clean and user-friendly online shopping experience with features such as product search, category filtering, shopping cart management, wishlist, and a checkout process.

This project demonstrates front-end web development concepts including responsive layouts, DOM manipulation, Local Storage, and interactive user interfaces.

**Features**
🏠 Responsive Home Page
🛍️ Product Listing with Images
🔍 Product Search
📂 Category Filter
⭐ Product Ratings
❤️ Wishlist Toggle
🛒 Add to Cart
❌ Remove from Cart
💾 Cart Data Stored using Local Storage
💳 Checkout Page
✅ Order Success Page
ℹ️ About Page
📞 Contact Page
📱 Mobile Responsive Design
🎨 Modern User Interface

## 💡 Key Functionalities

### 🛍️ Product Browsing
- Displays products with images, prices, ratings, and discount badges.
- Provides a clean and responsive product catalog.

### 🔍 Product Search
- Allows users to search products instantly by name using the search bar.

### 📂 Category Filtering
- Filters products by categories such as Electronics, Fashion, and Accessories.

### ❤️ Wishlist
- Users can mark products as favorites by toggling the wishlist icon.

### 🛒 Shopping Cart
- Add products to the shopping cart.
- Remove products from the cart.
- Automatically updates the total price and item count.

### 💾 Local Storage
- Saves cart items using the browser's Local Storage so that the cart remains available after refreshing the page.

### ✅ Order Confirmation
- Displays a success page after the order is placed and clears the cart.

### 📱 Responsive Design
- Optimized for desktops, tablets, and mobile devices using responsive CSS.

### 🎨 User-Friendly Interface
- Modern and interactive UI with smooth hover effects, attractive product cards, and easy navigation.
